# 🚀 Início Rápido - Barbearia Nativa

## ⚡ Para começar AGORA:

### 1. Instalar dependências
```bash
npm install
```

### 2. Inicializar banco de dados
```bash
npm run init-db
```

### 3. Iniciar servidor
```bash
npm run dev
```

### 4. Acessar o sistema
- **Site**: http://localhost:3000
- **API**: http://localhost:3000/api
